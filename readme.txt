== Description ==

MME Real Estate is a WordPress plugin developed by the Multimedia Efectiva team. MME Real Estate allows you to simply and efficiently manage a catalog of properties for rent or sale.

When programming this plugin we wanted to keep it simply, without complications, so you will not find excessive configurations or features that you will never use. Works on most popular WordPress themes.

Visit [multimediaefectiva.com/mme-real-estate-plugin-bienes-raices/](https://multimediaefectiva.com/mme-real-estate-plugin-bienes-raices/) for more information, in English and Spanish.

Do you have any suggestion? Would you like to incorporate a feature or adapt it to your operation? Do not hesitate to ask, we will be happy to assist you.
